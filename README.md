## Titulo 
Actividad 1: CRUD con room (Lista de tareas)

## Descripcion: 

App que permita agregar, editar, eliminar y listar tareas.

## Pasos de instalacion y ejecucion:

1.- Clonar el repositorio con:

      git clone https://github.com/ItsMazita/ListaDeTareas_A1.git

2.- Abrir el proyecto en Android Studio

3.- Configurar un dispositivo de prueba

## Estructura de la base de datos

| Campo          | Tipo     | Descripción                     |
|----------------|----------|---------------------------------|
| id             | int (PK) | Identificador único             |
| taskTitle      | String   | Título de la tarea              |
| taskDescription| String   | Descripción opcional            |
| completed      | boolean  | Estado de la tarea              |
| createdAt      | String   | Fecha de creación (dd/MM/yyyy)  |
